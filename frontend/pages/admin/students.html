Bulk Import Instructions

Use readable labels for printed/image masterlists and CSV files. Exact field keys are also accepted in CSV headers.

Printed/Image Masterlist: Student Labels

These are the labels the scanned image should show clearly.

Student Number First Name Middle Name Last Name Email Academic Level Department Course/Strand Year/Grade Section

Printed/Image Masterlist: Parent Labels

Parent fields are optional, but these labels help OCR find them.

Parent First Name Parent Middle Name Parent Last Name Parent Email Parent Phone Number Relationship Parent Address

CSV Student Header Keys

These exact keys are accepted, but readable labels like Student Number and Course/Strand work too.

studentNumber firstName middleName lastName email academicLevel department program yearLevel section username password status

`username`, `password`, and `status` are CSV-only optional fields. They are not expected in scanned images.

CSV Parent Header Keys

These exact keys are accepted, but readable labels like Parent First Name and Parent Phone Number work too.

parentFirstName parentMiddleName parentLastName parentEmail parentPhoneNumber parentRelationship parentUsername parentPassword parentAddress

Minimum student information

For a student row to import cleanly, provide at least:

Student Number First Name Last Name Academic Level

Course/Strand, Year/Grade, and Section are strongly recommended because they are used for filtering and records.

Minimum parent information

To create or link a parent during import, provide at least:

Parent First Name Parent Last Name Parent Phone Number

If parent email or phone already exists, the system links the existing parent instead of creating a duplicate.

Recommended CSV header

Student Number,First Name,Middle Name,Last Name,Email,Academic Level,Department,Course/Strand,Year/Grade,Section,Username,Password,Status,Parent First Name,Parent Middle Name,Parent Last Name,Parent Email,Parent Phone Number,Relationship,Parent Username,Parent Password,Parent Address

Printed masterlist tip

For image scanning, readable column titles are better than code-style keys. Use labels such as Student Number, Academic Level, Course/Strand, Year/Grade, Section, Parent Phone Number, and Parent Email. The scanned values still appear in the review popup for manual correction before import.
